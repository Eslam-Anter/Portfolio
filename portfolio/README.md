## Customization

### Colors
Edit CSS variables in `style.css`:
```css
:root {
    --navy: #0A1A2F;
    --teal: #0FA3B1;
    --sand: #F5EEDC;
    --white: #FFFFFF;
    --grey: #E6EBEF;
}
```

### Typography
Change font sizes in the CSS variables section or modify the font family links.

### Content
Edit all text content in `index.html` directly.
